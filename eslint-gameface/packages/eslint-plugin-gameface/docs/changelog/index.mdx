## 1.1.0 :badge[Latest]{variant="note"}

<Badge text="Feature" variant="success" size="small" /> Added support for linting `.vue` files.

<Badge text="Feature" variant="success" size="small" /> Added support for linting `.svelte` files.

:::note
Currently plugin supports Vue 3 single file components and Svelte 4/5 `.svelte` SFCs.
:::

## 1.0.0

<Badge text="Feature" variant="success" size="small" /> First release of ESLint support for Gameface projects: HTML, CSS, SCSS, JavaScript, JSX, TypeScript, and TSX.

<Badge text="Feature" variant="success" size="small" /> Recommended flat config preset to catch unsupported CSS, HTML, data binding, SVG, and JS API usage before runtime.
