## 0.0.1 :badge[Latest]{variant="note"}

<Badge text="Feature" variant="success" size="small" /> Initial release of `create-gameface-app`.

<Badge text="Feature" variant="success" size="small" /> Interactive scaffolding of Gameface projects from the `gameface-ui`, `solid`, `react`, `preact`, `vue`, and `svelte` templates.

<Badge text="Feature" variant="success" size="small" /> Command-line arguments for non-interactive use: a project name positional plus `--template`, `--yes`, and `--help` flags.

<Badge text="Feature" variant="success" size="small" /> Optional automatic installation of dependencies with the detected package manager once scaffolding completes.
