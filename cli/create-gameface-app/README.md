# create-gameface-app

A command-line tool that scaffolds a new Gameface project from a starter template. It downloads the template of your choice, optionally installs its dependencies, and leaves you with a project that is ready to run.

```bash
npm create gameface-app
```

You can read the full documentation for the CLI [here](https://frontend-tools.coherent-labs.com/create-gameface-app/getting-started/).
