Gameface Frontend Tools

Discover the complete suite of tools designed to streamline the creation and testing of game user interfaces with Gameface.

Gameface E2E

Gameface E2E is a powerful tool for testing your game user interfaces. It provides a robust framework for writing end-to-end tests, ensuring that your UI behaves as expected across different scenarios.

Check the Gameface E2E documentation

Interaction manager

The Interaction Manager is a utility that simplifies the process of managing user interactions in your game UI. It provides a set of tools to handle gamepad controls, spatial navigation, drag and drop, resizing, rotating, and panzoom functionalities.

Check the Interaction manager documentation

Gameface Vite Plugin

The Gameface Vite plugin is designed to enhance your development experience when using SolidJS with Gameface.

Check the Gameface Vite plugin documentation

Vite Gameface Style Transformer

A framework-agnostic Vite plugin and UnoCSS preset designed specifically for Coherent Gameface.

Check the Vite Gameface Style Transformer plugin documentation

Data binding autocomplete

The Data Binding Autocomplete vscode extension enhances your development experience by providing autocompletion for Gameface data bindings. It helps you quickly reference and bind data properties, improving productivity and reducing errors.

Check the extension’s documentation

ESLint plugin for Gameface

Catch unsupported CSS, HTML, data binding, SVG, and JavaScript patterns in your UI before runtime using ESLint plugin for Gameface.

Check the ESLint plugin documentation
