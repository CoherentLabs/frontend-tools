## 2.0.0 :badge[Latest]{variant="note"}

<Badge text="Change" variant="danger" size="small" /> Renamed `gf.moveMouse` method to `gf.mouseMove` for consistency with other mouse-related methods (breaking).

<Badge text="Feature" variant="success" size="small" /> Added `getParent` methods to both the `gf` object and the `DOMElement` object for retrieving the parent element of a specified element or the current DOM element, respectively.

## 1.1.1

<Badge text="Fix" variant="danger" size="small" /> Fixed tests exit code when tests fail. Previously, the exit code was always 0, which could lead to confusion when running tests in CI/CD pipelines.

## 1.1.0

<Badge text="Feature" variant="success" size="small" /> Introduced new assertion methods for the `DOMElement` object, leveraging `gf.retryIfFails` to ensure reliable validation of element states.

The newly added methods include:

* [element.waitForAttributes](/e2e/api/domelement/assertmethods/waitforattributes/)
* [element.waitForClasses](/e2e/api/domelement/assertmethods/waitforclasses/)
* [element.waitForPositionOnScreen](/e2e/api/domelement/assertmethods/waitforpositiononscreen/)
* [element.waitForSize](/e2e/api/domelement/assertmethods/waitforsize/)
* [element.waitForStyles](/e2e/api/domelement/assertmethods/waitforstyles/)
* [element.waitForText](/e2e/api/domelement/assertmethods/waitfortext/)
* [element.waitForValue](/e2e/api/domelement/assertmethods/waitforvalue/)
* [element.waitForVisibility](/e2e/api/domelement/assertmethods/waitforvisibility/)
* [element.waitForVisibilityInScrollableArea](/e2e/api/domelement/assertmethods/waitforvisibilityinscrollablearea/)

<Badge text="Feature" variant="success" size="small" /> Added the [gf.scrollTo](/e2e/api/gf/scrollto/) method to scroll the page to a specific position.

<Badge text="Feature" variant="success" size="small" /> Added the [gf.scrollToTop](/e2e/api/gf/scrolltotop/) method to scroll the page to the top.

<Badge text="Feature" variant="success" size="small" /> Added the [gf.scrollToBottom](/e2e/api/gf/scrolltobottom/) method to scroll the page to the bottom.

## 1.0.2

<Badge text="Fix" variant="danger" size="small" /> Fixed the `element.scrollIntoView` method to correctly identify the nearest scrollable container when none is explicitly provided through the arguments.

## 1.0.1

<Badge text="Fix" variant="danger" size="small" /> Resolved a random error where the WebSocket failed to locate an element with a specific ID while the tests are running.

<Badge text="Fix" variant="danger" size="small" /> Resolved an issue where the player would unexpectedly stop when `gf` methods were used without providing a required element selector.

<Badge text="Fix" variant="danger" size="small" /> Fixed the missing `core\commands\setup-mocha.js` file error when executing the `npx gameface-e2e`.

## 1.0.0

<Badge text="New" variant="tip" size="small" /> Official release of the Gameface E2E testing framework.

<Badge text="New" variant="tip" size="small" /> Added public documentation for the framework.

## 0.0.4

<Badge text="New" variant="tip" size="small" /> Introduced support for testing data bindings in the UI.

<Badge text="New" variant="tip" size="small" /> Added additional examples of how to test data bindings in the UI.

<Badge text="Feature" variant="success" size="small" /> Added methods to test data bindings by creating and updating models.

<Badge text="Feature" variant="success" size="small" /> Added methods to test data bindings by triggering and listening to engine events.

## 0.0.3

<Badge text="New" variant="tip" size="small" /> Introduced support for UI interactions.

<Badge text="New" variant="tip" size="small" /> Added additional examples for keyboard and mouse interactions.

<Badge text="Feature" variant="success" size="small" /> Enabled support for keyboard and mouse interactions.

<Badge text="Feature" variant="success" size="small" /> Added methods for scrolling elements into view within scrollable containers.

<Badge text="Feature" variant="success" size="small" /> Enabled support for spawning multiple test processes.

<Badge text="Feature" variant="success" size="small" /> Added runtime script execution support via the `gf.executeScript` method.

## 0.0.2

<Badge text="New" variant="tip" size="small" /> Added more test examples in the `examples` folder of the `gameface-e2e` npm package.

<Badge text="New" variant="tip" size="small" /> Introduced the `gf` object and `DOMElement` object for UI interactions. The `gf` object provides global utilities, while the `DOMElement` object represents specific DOM elements for interaction.

<Badge text="Feature" variant="success" size="small" /> Added methods to access UI elements.

<Badge text="Feature" variant="success" size="small" /> Added methods to validate UI elements.

<Badge text="Feature" variant="success" size="small" /> Introduced navigation to specific URLs via `gf.navigate(url)`.

## 0.0.1

Initial release of the e2e testing framework.

<Badge text="New" variant="tip" size="small" /> Enabled the framework to spawn the Gameface player and execute tests.

<Badge text="New" variant="tip" size="small" /> Added configuration file support, allowing users to specify the path to `Player.exe` and test files.
