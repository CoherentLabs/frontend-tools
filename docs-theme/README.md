# `coherent-docs-theme` 🎨

Theme for all the coherent documentations

## Package

If you are looking for the Starlight plugin package, you can find it in the [`packages/coherent-docs-theme/`](/packages/coherent-docs-theme/) directory.

## Project structure

This project uses pnpm workspaces to develop a single Starlight plugin from the `packages/coherent-docs-theme/` directory. A Starlight documentation site is also available in the `docs/` directory that is also used for testing and demonstrating the Starlight plugin.
